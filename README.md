# Crossmodal Decoding of Emotional Faces: Perception vs. Imagery

Neural alignment across perception and imagery of emotional faces,
analyzed via MVPA and RSA methods.

## Status

**Manuscript in preparation.** The code in this repository is shared
for transparency and development purposes. A formal open-source license
will be added upon publication. Until then, please contact the author
for any use beyond inspection.

## Contents

Analysis pipeline is located in
[`KiHEON_LEE_CRSSMDAL_DECODE_RSA_PTnIT_260320/`](./KiHEON_LEE_CRSSMDAL_DECODE_RSA_PTnIT_260320/),
including preprocessing, first-level GLM, group-level univariate analysis,
cross-task RSA, cross-task decoding, and neuro-behavioral correlation
analysis.

See the [subfolder README](./KiHEON_LEE_CRSSMDAL_DECODE_RSA_PTnIT_260320/README.md)
for pipeline details.
